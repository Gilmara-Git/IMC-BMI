## Objectives

- Grab the values that the user has typed for weight (Kg) and height (Cm) and apply the IMC(BMI) formula which is weight / (height/100 \* height/100 ).

## Languages/Technologies

- Html5 (Hypertext)
- Css3 (Cascading Style Sheet)
- Javascript
